# Agro Forte - Futuro Sustentável

## Descrição do Projeto
O projeto **Agro Forte, Futuro Sustentável** tem como objetivo apresentar práticas agrícolas que equilibram produção e preservação ambiental, promovendo o desenvolvimento sustentável e conscientização sobre a importância do meio ambiente.

## Tecnologias Utilizadas
- **HTML**: Estrutura semântica do site, incluindo seções, navegação e formulário de contato.
- **CSS**: Layout responsivo, cards de projetos, botões interativos e estilização geral.
- **JavaScript**: 
  - Mostrar e ocultar projetos extras dinamicamente.
  - Validação do formulário de contato com feedback visual.
  - Manipulação do DOM para melhorar a interação do usuário.
